# Furniture Quotation Management System - Backend

Production-ready Node.js backend with TypeScript, Express, PostgreSQL, and Sequelize ORM.

## Tech Stack

- **Runtime:** Node.js + TypeScript
- **Framework:** Express.js
- **Database:** PostgreSQL
- **ORM:** Sequelize with migrations
- **Auth:** JWT (Access + Refresh tokens)
- **Validation:** Zod
- **Email:** Nodemailer
- **Security:** Helmet, CORS, XSS protection, HPP, Rate limiting

## Prerequisites

- Node.js >= 18
- PostgreSQL >= 14
- npm >= 9
